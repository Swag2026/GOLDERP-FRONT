💾 Data Backup
All data is stored in your browser's localStorage — it persists through page refreshes and browser restarts. Download a JSON backup regularly to avoid data loss if the browser cache is cleared.
Google Sheets Connection
Currently running in offline mode — data is saved locally. Connect a Google Apps Script Web App URL to sync entries directly to a Google Sheet in real time.
Roles & Permissions Reference
Cashier
Can add new entries and view History. Cannot edit, cancel, or access Analytics / Reports / Settings.

Supervisor
Full access to ledger, can edit and cancel entries (mandatory note required). Can view Analytics and Reports. Cannot access Settings.

Admin
Full access to everything including Settings, user management, backup/restore, and data reset. Edit/cancel without mandatory note.
⚠ Danger Zone
Warning: Resetting all data will delete every entry and restore the seed demo data. This action cannot be undone. Download a backup first.
